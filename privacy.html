Privacy Policy

Last updated: July 23, 2026

1. Introduction

This Privacy Policy describes how the Pinterest Auto Uploader application ("the App") handles data when used to upload media files to Pinterest.

2. Data Collection

The App collects and processes the following data:

  • Image and video files from the designated local folder (C:\Users\Lego\Desktop\Uploader)
  • File metadata (filename, file size, upload timestamps)

3. Data Usage

The App uses collected data for:

  • Uploading media files to the user's own Pinterest account
  • Tracking uploaded files to prevent duplicates
  • Logging upload activity for troubleshooting

4. Data Sharing

The App does NOT:

  • Share data with third parties
  • Sell or monetize user data
  • Collect data from other users
  • Track user behavior outside the App

5. Data Storage

  • All data is stored locally on the user's computer
  • Upload history is saved in a local JSON file
  • No data is transmitted to external servers except Pinterest API

6. Pinterest API Usage

The App uses the Pinterest API to:

  • Create pins on the user's own Pinterest boards
  • Access is limited to the user's own Pinterest account
  • No other Pinterest users' data is accessed

7. Security

  • The App uses HTTPS for all API communications
  • Access tokens are stored locally and not shared
  • No sensitive data is logged

8. Children's Privacy

The App is not intended for use by children under 13 years of age.

This Privacy Policy may be updated from time to time. Changes will be reflected in the "Last updated" date above.
